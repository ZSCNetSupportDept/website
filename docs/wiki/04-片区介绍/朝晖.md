# 朝晖片区信息

集合点：朝晖楼门口

范围：朝晖苑

工具包位置：门口右手边，值班室对面的房间的桌子里面，和售货机和打印机在同一个房间
## 网络结构
### 宿舍内部
用户端口直连配线架，配线架连接到交换机，没有床位表格，需要手动寻线

对于24新生的宿舍已采用了FTTR架构
```mermaid 宿舍网络结构

graph LR
    subgraph 用户设备
    用户电脑
    end
    
    subgraph  宿舍线路 
    用户电脑--**用户的网线**-->端口
    end
    
    subgraph 机房线路
    端口--**墙线**--->配线架-->交换机
    end
    
    交换机-->yys[(校园网)]

```
