联通特定

信息

在维修联通问题之前，推荐阅读这篇文章

提示

在维修与错误码相关的问题时，请善用AI和互联网

📄️ 账号不正确

老登修网请注意：

📄️ 故障代码1062

”服务尚未启动。“(The service has not been started.)

📄️ 错误代码619

"无法连接到远程计算机。"(A connection to the remote computer could not be established...)

📄️ 故障代码629

“远端计算机已中断端口连接。”(The port has been disconnected by the remote computer

📄️ 错误代码633

"调制解调器(或其他连接设备)已在使用中或没有正确配置。"(The modem (or other connecting device) is already in use or is not configured properly.)

📄️ 故障代码651

"调制解调器(或其他连接设备)报告了一个错误。"(The Modem (or other connecting device) has reported an error.)

📄️ 故障代码678

"远程计算机没有响应。"(The remote computer did not respond.)

📄️ 故障代码691

"访问被拒绝，因为用户名或密码在域上无效。"(Access denied because the username or password is invalid on the domain.)

📄️ 错误代码692

"调制解调器(或其他连接设备)中存在硬件故障。"(There was a hardware failure in the modem (or other connecting device).)

📄️ 故障代码711

"远程访问连接管理器服务无法启动。"(The Remote Access Connection Manager service could not start...)

📄️ 故障代码720

"无法建立到远程计算机的连接。"(A connection to the remote computer could not be established. You might need to change the network settings for this connection.)

📄️ 故障代码734

"PPP链接控制协议被终止。"(The PPP link control protocol was terminated.)

📄️ 错误代码769

"无法访问目标。"(The specified destination is not reachable.)

📄️ 错误代码813

“PPPoE拨号连接处于活动状态。”(another connection is using....)
