# 概述

本文档描述的“报修系统”，是网维历史上第三版报修系统，代号"wts3",采用前后端分离架构。与前任系统相同，该系统在用户前端上主要依托于微信公众号(服务号)作为流量入口，亦可以作为网页访问，其后端采用Go语言编写，使用的数据库为PostgreSQL

## 技术栈
### 前端
- 开发框架：SvelteKit
- UI框架：Carbon Design System (Carbon Design Svelte)
- HTTP框架：Axios

### 后端
- 语言：Go
- HTTP框架：Echo
- 数据库：PostgreSQL
