# 表结构
![illustration](/img/devdocs/wts3-dbDesign-first-draft.png)

## schema:data
data中存放的是系统运行的数据
### data.students
学校所有学生的姓名和学号，系统依赖于这个表运行，记录由管理人员负责插入，从学校那里拿数据，每年一般都是由开发组组长负责
|列名|类型|属性|配置|备注|
|-|-|-|-|-|
|sid|text|主键||学号|
|name|text|||姓名|
## shcema:wts
报修系统的主数据库

### wts.users
报修系统的用户，在注册时需要与data.students校验
|列名|类型|属性|配置|备注|
|-|-|-|-|-|
|sid|text|主键，外键->data.students.sid|on delete restrict,on update cascade|用户的学号|
|phone|text|unique||手机号|
|block|wts.block|||住哪栋|
|room|text|||住哪间|
|isp|wts.isp|||运营商|
|account|text|unique||账号|
|wx|text|unique,not null||微信（OpenID）|
