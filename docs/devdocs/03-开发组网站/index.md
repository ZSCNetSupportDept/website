# 开发组网站
就是你所看到的网站；这个网站大概包含了三个部分：

1. 网维的新wiki

2. 开发组相关文档

3. 一个博客系统，可以用来放独立的文章



因为感觉没有必要搭建两个基础设施，所以把wiki和开发组文档放在一起，这样也比较方便运维

网站采用[Docusaurus](https://www.docusaurus.io)框架，详细文档移步官网查看

**项目地址**：https://github.com/ZSCNetSupportDept/website
